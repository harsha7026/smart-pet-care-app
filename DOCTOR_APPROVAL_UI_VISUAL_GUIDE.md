### Desktop (> 1024px)
```
Modal width: 900px (max)
Sections side-by-side where possible
Full detail grid: 2 columns
Vaccine items: 4 per row
```

### Tablet (768px - 1024px)
```
Modal width: 90% of screen
Sections stack vertically
Detail grid: 2 columns
Vaccine items: 2 per row
```

### Mobile (< 768px)
```
Modal width: 90% of screen, full height with padding
All sections stack vertically
Detail grid: 1 column
Vaccine items: 1 per row (full width)
Buttons: Full width in modal footer
Time input: Full width
```
